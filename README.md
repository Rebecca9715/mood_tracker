# 心情追踪小程序

一个帮助用户记录和分析日常心情的微信小程序。通过 AI 助手提供情感支持和建议，让用户更好地了解和管理自己的情绪。

## 主要功能

- 📝 心情记录：随时记录当下的心情和想法
- 🤖 AI 分析：智能分析情绪变化，提供个性化建议
- 📊 数据统计：直观展示心情变化趋势
- 🎵 音乐推荐：根据心情智能推荐音乐
- 📅 每日分析：生成每日心情分析报告

## 技术栈

- 微信小程序原生开发
- 深度学习模型：DeepSeek
- 本地数据存储
